# 提示词：图片描述 (figure_caption) — 多模态

- **用途**：用 VLM 为抠出的图生成中文描述，连同原 caption 一起向量化，让"那张图在讲什么"可检索；原图存 MinIO 答案回显。
- **调用位置**：parsing/indexing，处理 `block_type=figure`，调 VLM。
- **输入**：图片(MinIO) + `{caption}`
- **输出**：中文图片描述。

## Prompt（多模态，图 + 文）
```
这是一篇学术论文里的插图，原始图注：{caption}。
请用中文描述：图中展示了什么（结构/流程/坐标轴/对比对象/关键趋势）。
客观描述，不臆造数据，不超过 120 字。
```

## 优化建议
- 流程图/架构图重点描述"模块与关系"；曲线图重点描述"轴与趋势"。
- 描述 + 原 caption 拼接后入 content_zh，提升图检索召回。
