# 提示词：Self-RAG 自检 (self_rag_reflect)

- **用途**：答案生成后自我反思——每个结论是否有出处支撑，无依据的标注/删除，抑制幻觉。
- **调用位置**：chat_agent 生成后（进阶，可选）。
- **输入**：`{answer}`、`{context}`
- **输出**：JSON（是否有未支撑结论 + 修订建议）。

## Prompt
```
核查下面的【答案】是否每个事实性结论都能在【参考资料】中找到支撑。

【参考资料】：
{context}

【答案】：
{answer}

只输出 JSON：
{"faithful": true|false,
 "unsupported": ["列出无依据的句子"],
 "revised_answer": "若有无依据内容,给出删除/标注后的修订版;否则原样返回"}
```

## 优化建议
- 高准确性场景（医疗/学术）建议开启；闲聊/常识可跳过省延迟。
- 可只标注不删除，把"此处缺乏出处"提示给用户，更透明。
