# 提示词：查询重写 (query_rewrite)

- **用途**：把口语化/残缺/有歧义的提问，改写成清晰、适合向量编码的检索式。
- **输入**：`{question}`、`{history}`
- **输出**：一行改写后的查询。

## Prompt
```
你是检索查询优化器。把用户提问改写成清晰、完整、无歧义、适合语义检索的句子。
要求：补全指代(结合历史)、去除口水词、保留专业术语、不改变原意、不要新增信息。

对话历史：{history}
原始提问：{question}

只输出改写后的查询，一行，不要解释。
```

## 优化建议
- 追问场景务必结合历史补全"它/这个"等指代。
- 别过度改写导致语义漂移；改写前后语义一致是底线。
