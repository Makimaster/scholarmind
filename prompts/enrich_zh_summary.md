# 提示词：入库中文增强 (enrich_zh_summary) — 文档增强

- **用途**：入库时为每个英文 chunk 生成中文摘要 + 关键词，写入 `content_zh`，让中文 query 同语言命中。跨语言召回的杀手锏。
- **调用位置**：indexing 服务，向量化前。
- **输入**：`{chunk_text}`（英文原文）、`{section}`
- **输出**：JSON（摘要 + 关键词）。

## Prompt
```
下面是一篇英文学术论文的片段（章节：{section}）。请用中文：
1. 生成 1-2 句忠实摘要（不超过 80 字，不臆造）；
2. 抽取 3-6 个中文关键词（含中文译名 + 必要的英文术语原词）。

片段：
{chunk_text}

只输出 JSON：{"summary_zh": "...", "keywords_zh": ["...", "..."]}
```

## 优化建议
- 表格/公式片段单独用 table_summary / 公式解释提示词，别用本提示词硬套。
- 摘要务必忠实，禁止补充原文没有的信息（否则污染检索）。
