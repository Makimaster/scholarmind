# 提示词：表格摘要 (table_summary) — 小-大检索

- **用途**：大表不塞一个 chunk；用本提示词生成"表讲了什么"的中文摘要做检索（子），整表 HTML 存 `doc_blocks`（父），命中后取回。
- **调用位置**：indexing，处理 `block_type=table`。
- **输入**：`{table_html}`、`{caption}`
- **输出**：中文摘要（入 content_zh）。

## Prompt
```
下面是论文中的一张表格（HTML），标题：{caption}。
用中文概括：这张表对比了什么、有哪些指标/列、最关键的结论或最大值在哪。
不超过 100 字，忠实于数据，不臆造。

表格：
{table_html}

只输出中文摘要。
```

## 优化建议
- 列名很多时，摘要里点出关键列即可，细节由命中后取回的整表提供。
- 数值类问题（"最高/趋势"）靠取回整表给 LLM 算，别指望摘要算。
